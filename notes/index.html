Rio

Terminal moderno, bonito e acelerado por GPU, escrito em Rust por um brasileiro. Suporta ligaturas, abas, split panes e configuração via TOML.

GitHub: https://github.com/raphamorim/rio

Tangent

Ferramenta local de notas/conhecimento pessoal inspirada no Obsidian. Suporta [[Wiki Links]], markdown customizado, mapa de navegação 2D, query language própria e visualização de cards.

Site: https://www.tangentnotes.com/

GitHub: https://github.com/suchnsuch/Tangent

NotebookLM MCP CLI

Interface de linha de comando que integra o Google NotebookLM com o protocolo MCP (Model Context Protocol). Permite buscar, consultar e gerenciar notas do NotebookLM via terminal/LLMs.

GitHub: https://github.com/jacob-bd/notebooklm-mcp-cli

Spotify Player

Cliente TUI completo para Spotify, escrito em Rust. Navegue, busque, toque músicas e gerencie playlists direto do terminal, com suporte a streaming e cache local.

GitHub: https://github.com/aome510/spotify-player
